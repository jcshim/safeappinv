# SafeApp
안전앱 개발

## AppInv의 기초
1) 앱인벤터 활용법
2) 비서 앱 (음성인식 및 흔들면 말하기)-버튼,가속도 센서,음성인식,음성합성
3) 사진찍어 카톡 보내기-카메라센서, 소셜네트워크 통신
4) 멀티미디어 응용(동물울음소리)-버튼,영상 미디어, 음성 미디어
* 아이콘 만들기
6) 두더지 잡기(aia 활용하기)
* 개별 프로젝트

## 고급 응용과 와이 파이
1) 낙서장 만들기-팔레트, 색상, 슬라이드바, 나눗셈
2) 프로세싱 소개 및 예제 실행
3) 서버와 Wifi 통신하기 
* 팀프로젝트 기획 및 계획서
